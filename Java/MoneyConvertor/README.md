# MoneyConvertor.java

MoneyConvertor类能够将小写数字金额转换为中文大写金额。

## How to use

``` java

    String money = "368.503";
    MoneyConvertor moneyConvertor = new MoneyConvertor(money);
    String result = moneyConvertor.convert();
    System.out.println(result);

```
